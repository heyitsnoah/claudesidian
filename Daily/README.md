# Daily Notes

Quick daily captures and journal entries.

## Purpose

A space for daily reflections, task tracking, and quick notes that don't fit elsewhere.

## Naming Convention

Use date-based naming:
- `YYYY-MM-DD.md` (e.g., `2025-01-25.md`)
- Or `YYYY-MM-DD - Topic.md` for themed days

date: YYYY-MM-DD
