# Templates

Reusable note templates for various purposes.

## Available Templates

Create templates for:
- Daily notes
- Meeting notes
- Project planning
- Literature notes
- Permanent notes
- Weekly reviews
- Research summaries

## Using Templates

In Obsidian, you can use templates via:
1. Core Templates plugin
2. Templater plugin (more powerful)

## Template Variables

Common variables (Templater syntax):
- `<% tp.date.now("YYYY-MM-DD") %>` - Current date
- `<% tp.file.title %>` - File name
- `<% tp.file.cursor() %>` - Cursor position

## Tips

- Keep templates simple and adaptable
- Include prompts for sections you often forget
- Use consistent frontmatter across templates
- Review and refine templates periodically
