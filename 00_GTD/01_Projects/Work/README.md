# Work Projects

Work-related projects and initiatives.

## Creating a Project

A project is any outcome requiring more than one action step.

### Template

```markdown
# Project: [Name]

## Outcome
What does "done" look like?

## Stakeholders
Who is involved?

## Next Actions
- [ ] First concrete step
- [ ] Second step

## Supporting Materials
- [[Related notes]]

## Log
- YYYY-MM-DD: Started project
```

## Tips

- Keep project notes focused on outcomes and actions
- Link to supporting materials in other folders
- Update [[../_PJM|Project Dashboard]] when adding new projects
- Move completed projects to Archive
