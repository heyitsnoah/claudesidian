## Project Quick Add

To create a new project:
1. Create a note in `Personal/` or `Work/`
2. Define the desired outcome
3. Identify the next action
4. Add to the table above

## Project Template

```markdown
# Project: [Name]

## Outcome
What does "done" look like?

## Why
Why is this important?

## Next Actions
- [ ] First concrete step

## Supporting Materials
- [[Related notes]]

## Status
- Started: YYYY-MM-DD
- Target: YYYY-MM-DD
- Status: Active / On Hold / Completed
```
