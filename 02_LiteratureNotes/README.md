# Literature Notes

Notes taken while engaging with external sources.

## Purpose

Literature notes capture ideas from books, articles, videos, podcasts, and other content in your own words.

## Characteristics

- **Source-based**: Tied to a specific piece of content
- **In your own words**: Paraphrase, don't just copy
- **Include citations**: Always note the source
- **Selective**: Capture what's relevant to your thinking

## Structure

Each literature note should include:

```markdown
---
source: [Book/Article/Video title]
author: [Author name]
date_consumed: YYYY-MM-DD
type: book/article/video/podcast
---

## Key Ideas

- Main point 1
- Main point 2

## Quotes

> "Direct quote" (p. XX)

## My Thoughts

[Your reactions, questions, connections]

## Related

- [[Permanent Note 1]]
- [[Permanent Note 2]]
```

## Processing

1. Create literature notes while consuming content
2. Review and extract key insights
3. Transform insights into permanent notes
4. Link to existing permanent notes
5. Archive or keep for reference
